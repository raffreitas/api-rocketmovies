
## API Rocketmovies 


API desenvolvida utilizando NodeJS

### Foi desenvolvido algumas funcionalidades como

- Criptografia de senhas;
- Validação de e-mail;
- Aplicar o cascade para garantir que uma tag será excluída caso o usuário opte por excluir a nota.


## Instalação

```bash
  npm run dev
  nmp run migrate
```
